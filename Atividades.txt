Atividade para casa

1 - Clonar o repositório lógico de programação para a máquina local.

2 - Criar um arquivo no repositório local com o nome
casa.txt

3 - No arquivo casa.txt insira a seguinte informação: "criei este arquivo em casa e farei o commit de saus alterações
e em seguida o push para o repositorio remoto."

4 - Utilizando o github desktop faça o commit do arquivo das alterações.

5 - Agora faça o push orign;

6 - Acesse o repositório remoto e visualize o arquivo casa.txt
